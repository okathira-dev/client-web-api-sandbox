Web API を使った遊び場

これはWeb APIを使ったサンドボックスプロジェクトです。
様々なWeb APIを試すための実験的な実装が含まれています。

プロジェクト一覧

  • クロマティックボタンアコーディオン - 鍵盤でクロマティックボタンアコーディオンを演奏できるウェブアプリ
  • データモッシング PoC - WebCodecsを使ったデータモッシング実験
  • データモッシング React App WIP - WebCodecsを使ったデータモッシングのReact実装
  • 線形合同法乱数予測ツール - 線形合同法による乱数を予測するツール
  • Computation of Tears WIP - Tears of Overflowed Bits (by eau. / La Mer ArtWorks)の再現
  • 心拍モニター - スマートフォンのカメラを使用して心拍数を測定するアプリ
